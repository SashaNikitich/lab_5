## Опис реалізації

Програма написана на мові **C#** (.NET 9) і демонструє застосування принципів **успадкування, поліморфізму та інкапсуляції** для моделювання музичного альбому.

### 1. Абстрактний клас `Track`

* Базовий клас для всіх типів музичних композицій.
* Містить спільні властивості: `Title` (назва), `Duration` (тривалість) та `Genre` (жанр).
* Включає валідацію даних: назва не може бути порожньою, а тривалість має бути більшою за нуль.
* Перевизначає метод `ToString()` для гарного форматування інформації про трек.

### 2. Ієрархія жанрів

Реалізовано конкретні класи, що успадковуються від `Track`:

* **`PopMusic`**: автоматично встановлює жанр "Pop".
* **`RockMusic`**: автоматично встановлює жанр "Rock".
* **`MetalMusic`**: автоматично встановлює жанр "Metal".

### 3. Клас `Album`

* Контейнер для зберігання колекції треків (`List<Track>`).
* **AddTrack**: метод додавання треків з перевіркою на `null`.
* **GetTotalDuration**: обчислює сумарний час усіх композицій за допомогою LINQ.
* **SortByStyle**: виконує багаторівневе сортування: спочатку за **жанром**, потім за **тривалістю**, і наостанок за **назвою**.
* **FindByDuration**: фільтрує треки, тривалість яких потрапляє у заданий користувачем проміжок .

### 4. Обробка помилок

* Програма використовує блоки `try-catch` для перехоплення виключних ситуацій (наприклад, некоректна тривалість треку або спроба додати порожній об'єкт).

## Структура проєкту

* `Track.cs` — абстрактний базовий клас композиції.
* `PopMusic.cs`, `RockMusic.cs`, `MetalMusic.cs` — похідні класи конкретних жанрів.
* `Album.cs` — клас-менеджер для роботи з колекцією треків.
* `Program.cs` — точка входу, демонстрація створення альбому, сортування та пошуку.
